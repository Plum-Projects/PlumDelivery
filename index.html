PlumDelivery

Advanced Delivery & Quest Event System for Minecraft Servers

Get Started →Configuration

⏰ Timed Delivery Events

Daily (24h), Weekly (7d), and Monthly (30d) delivery cycles with automatic countdown timers and auto-restart support.

📦 Unlimited Categories

Create as many delivery categories as you want — Farming, Combat, Mining, and more with unique objectives.

🏆 Leaderboard System

Real-time in-game leaderboard with customizable top player count, displayed in beautiful GUI menus.

🎯 Progressive Goals

Set multiple point milestones per category with progressive reward commands that execute instantly.

💾 Multi-Database Support

Choose from RozsDBLite (file-based), SQLite, or MySQL/MariaDB with HikariCP connection pooling.

⚡ Folia Ready

Full compatibility with Folia for modern multi-threaded servers. Also supports Paper and Purpur.

🔗 PlaceholderAPI

Built-in PAPI placeholders for scoreboards, holograms, tab menus, and more.

🎁 Smart Rewards

Goal milestone rewards + placement rewards for top players. Works even for offline players!

🔧 100% Configurable

Every menu, item, lore, color, message, and behavior is fully customizable via YAML configuration.

Quick Start

1. Install the Plugin

Download PlumDelivery.jar and place it in your server's plugins/ folder.

2. Configure Categories

Edit plugins/PlumDelivery/config.yml to define your delivery categories with objectives and goals.

3. Start an Event

/delivery start daily

4. Let Players Compete!

Players open the menu with /delivery, choose categories, complete objectives, and compete on leaderboards.

Supports Minecraft 1.13 — 1.21.x | Paper, Purpur & Folia | Java 21+
