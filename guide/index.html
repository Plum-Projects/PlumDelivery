Getting Started

What is PlumDelivery?

PlumDelivery is a powerful and fully configurable delivery/quest event plugin for Minecraft servers. It brings a competitive edge to your server by letting players complete delivery tasks across customizable categories — collecting items and killing mobs — to earn points, climb leaderboards, and claim incredible rewards!

With support for Daily, Weekly, and Monthly delivery cycles, your server always has fresh content to keep players engaged.

Key Features

  • ⏰ Timed Delivery Events — Daily (24h), Weekly (7d), and Monthly (30d) cycles with automatic countdown timers
  • 📦 Unlimited Categories — Create as many categories as you want (Farming, Combat, Mining, etc.)
  • 🎯 Goal Milestones — Set multiple point thresholds per category with progressive rewards
  • 🏆 Leaderboard System — Real-time in-game leaderboard with customizable top player count
  • 🎁 Placement Rewards — Automatic reward distribution for top-N players when events end
  • 📊 Dual Action Types — Support for COLLECT (items) and KILL (mobs) objectives
  • 🔄 Auto-Restart — Events automatically restart after completion
  • 🖥️ Beautiful GUI Menus — Fully customizable inventory menus
  • 💾 Multi-Database — RozsDBLite, SQLite, and MySQL/MariaDB
  • ⚡ Folia Support — Full compatibility with Folia servers
  • 🔗 PlaceholderAPI — Built-in PAPI placeholders for scoreboards and holograms
  • 🌐 Multi-Language — Fully translatable messages with color code support
  • 🛡️ Offline Rewards — Top players receive rewards even if they're offline

Installation

  1. Download the latest PlumDelivery.jar from SpigotMC or GitHub
  2. Place the JAR file in your server's plugins/ folder
  3. Start (or restart) your server
  4. The plugin will generate default configuration files in plugins/PlumDelivery/
  5. Customize config.yml and messages.yml to your liking
  6. Use /delivery reload to apply changes without restarting

Quick Start

1. Configure Categories

Edit config.yml to define your delivery categories. Each category needs:

  • A name and icon
  • Required objects (items to COLLECT or mobs to KILL)
  • Goals with reward commands

2. Start an Event

/delivery start daily

This starts a daily delivery event. The countdown timer begins immediately.

3. Let Players Play

Players can open the delivery menu with:

/delivery

Or the Turkish alias:

/teslimat

4. Watch the Competition

Players will:

  • Choose a delivery type (Daily/Weekly/Monthly)
  • Select a category
  • Collect items or kill mobs
  • Submit items via the GUI
  • Earn points and unlock milestone rewards
  • Compete on the leaderboard

5. Rewards Distribution

When the timer expires (or you manually end the event), top players receive placement rewards automatically!

/delivery end daily
